# DM Toolbox — Character & Creature Concept Art Prompts

Reference prompts for generating character/creature concept art via **gpt-image-2** (or similar image models) using an uploaded sketch as a pose & design reference.

---

## How to use

1. Pick the campaign **setting**: Dark Sun, Ravenloft, or Generic Fantasy
2. Pick a **background** (1–4) within that setting
3. Copy the **Master Prompt** for the chosen setting
4. Paste the chosen **Setting** description in place of the `[user selects options 1 to 4 below]` line
5. The `Color palette` line is auto-resolved — each Setting option includes its own palette
6. Attach the reference sketch and submit

**Recommended size:** 4K Landscape (3840×2160) for best detail.
**Tip:** run 2–3 generations per prompt and cherry-pick — results vary, and sketch adherence improves with multiple attempts.

## Notes & gotchas

- **No artist names** — gpt-image-2 rejects prompts containing specific artist references. The "1980s fantasy role-playing game cover art / vintage fantasy paperback illustration" phrasing reliably produces the same aesthetic without tripping filters.
- **Lighting logic is consistent** across all 12 backgrounds: warm light source from camera-right, cool shadows on camera-left. This means the subject will look like the same character lit consistently across any setting/background combo.
- **Sketch fidelity** — the opening line ("preserve the character's pose, proportions, silhouette, gear, and key design details exactly as drawn") is critical. Without it, the model reinterprets the design.
- **If a background isn't landing** — regenerate. gpt-image-2 has meaningful variance between runs, especially for crowded scenes (royal court, city state) where it has to render many small elements.
